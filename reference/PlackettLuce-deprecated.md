# Deprecated functions in package PlackettLuce

The functions listed below are deprecated and will be defunct in the
near future. When possible, alternative functions with similar
functionality are also mentioned.

## Usage

``` r
grouped_rankings(rankings, index, ...)
```

## grouped_rankings

`grouped_rankings()` has been replaced by
[`group()`](https://hturner.github.io/PlackettLuce/reference/group.md).
